# Place & Trade Area Data Visualization

Interactive map application for visualizing place and trade area data using React, Material-UI, Deck.gl, and Mapbox.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- No API keys required! (Uses free MapLibre)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd next-js-case
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
